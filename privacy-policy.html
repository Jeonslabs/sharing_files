파일저장 개인정보처리방침

최종 업데이트: 2025년 12월 23일
시행일: 2025년 12월 23일

파일저장 앱(이하 "본 앱")은 사용자의 개인정보를 보호하고 소중하게 생각합니다. 본 개인정보처리방침은 본 앱이 사용자의 개인정보를 어떻게 수집, 사용, 보호하는지 설명합니다.

1. 개인정보의 수집

중요: 본 앱은 사용자의 개인정보를 서버로 수집하거나 전송하지 않습니다.

1.1 수집하는 정보

  • 문서 데이터: 사용자가 앱에 저장하는 이미지, 제목, 메모, 카테고리 정보
  • 앱 설정 정보: PIN 번호, 생체인증 설정, 카테고리 설정

1.2 정보 저장 위치

  • 모든 데이터는 사용자 기기 내부에만 저장됩니다
  • 외부 서버로 전송되거나 클라우드에 업로드되지 않습니다
  • 앱을 삭제하면 모든 데이터가 함께 삭제됩니다

2. 개인정보의 이용 목적

수집된 정보는 다음의 목적으로만 사용됩니다:

  • 문서 관리: 사용자가 저장한 문서를 기기 내에서 관리
  • 검색 기능: 사용자가 저장한 문서를 빠르게 찾을 수 있도록 지원
  • 보안 기능: PIN 및 생체인증을 통한 앱 접근 제어
  • 카테고리 관리: 문서를 체계적으로 분류 및 정리

3. 개인정보의 공유 및 제공

본 앱은 사용자의 개인정보를 제3자와 공유하거나 판매하지 않습니다.

3.1 공유 기능

  • 사용자가 명시적으로 "공유하기" 기능을 사용할 때만 선택한 문서가 다른 앱(예: 카카오톡)으로 전송됩니다
  • 이는 사용자의 직접적인 행동에 의한 것이며, 자동으로 공유되지 않습니다

4. 개인정보의 보안

본 앱은 사용자 정보를 보호하기 위해 다음과 같은 보안 조치를 취합니다:

4.1 앱 잠금 기능

  • PIN 인증: 4자리 PIN으로 앱 접근 제어
  • 생체 인증: 지문 또는 얼굴 인식으로 추가 보안
  • 재시도 제한: 잘못된 PIN 입력 시 일시적 접근 차단

4.2 데이터 보호

  • 모든 데이터는 Android 보안 샌드박스 내에 저장됩니다
  • 다른 앱에서 접근할 수 없도록 보호됩니다
  • 기기 암호화 기능이 활성화된 경우 함께 암호화됩니다

5. 권한 요청

본 앱은 다음 권한을 요청합니다:

5.1 카메라 권한

  • 목적: 문서 사진을 직접 촬영하기 위함
  • 사용 시점: 사용자가 "카메라로 촬영" 기능을 선택할 때만

5.2 저장소 권한

  • 목적: 갤러리에서 이미지를 선택하고 저장하기 위함
  • 사용 시점: 사용자가 이미지를 추가하거나 공유할 때

5.3 생체 인증 권한

  • 목적: 지문 또는 얼굴 인식으로 앱 잠금 해제
  • 사용 시점: 사용자가 생체 인증을 활성화했을 때

6. 사용자의 권리

사용자는 언제든지 다음 권리를 행사할 수 있습니다:

  • 데이터 조회: 앱 내에서 저장된 모든 문서 확인
  • 데이터 수정: 앱 내에서 문서 정보 수정
  • 데이터 삭제: 앱 내에서 개별 문서 또는 전체 데이터 삭제
  • 앱 삭제: 앱을 삭제하면 모든 데이터가 완전히 제거됩니다

7. 아동의 개인정보 보호

본 앱은 만 13세 미만 아동의 개인정보를 의도적으로 수집하지 않습니다. 만 13세 미만 아동이 앱을 사용하는 경우, 부모 또는 법정대리인의 동의가 필요합니다.

8. 외부 서비스

본 앱은 외부 분석 도구, 광고 네트워크, 또는 제3자 서비스를 사용하지 않습니다.

9. 개인정보처리방침 변경

본 개인정보처리방침은 법률 변경 또는 서비스 개선에 따라 수정될 수 있습니다. 중요한 변경사항이 있을 경우 앱 내 공지 또는 업데이트를 통해 알려드립니다.
